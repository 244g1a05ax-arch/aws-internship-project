# AWS Cloud Computing Internship Project

## Project Purpose
This repository documents Task 1 of my AWS Cloud Computing internship. It includes a cloud concepts report and evidence of AWS account setup and security configuration.

## Repository Structure
- `docs/cloud_concepts_report.md` — Report covering cloud deployment models, service models, AWS account security setup, and core AWS services.
- `docs/screenshots/` — Screenshots showing MFA setup, billing alarm configuration, IAM admin user creation, and core AWS service dashboards (EC2, S3, VPC).

## Setup Steps Covered
- Enabled MFA on the AWS root account.
- Created a Zero Spend Budget billing alarm.
- Created an IAM admin user for daily account use.
- Explored core AWS services: EC2, S3, and VPC.
